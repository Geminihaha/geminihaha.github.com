# Zombie Hunter — 개발 기록

## 개요

뱀파이어 서바이버 스타일의 웹 게임. 순수 HTML + Canvas + JavaScript로 제작됨.

- **원본 출처**: Ghost Hero 레포지토리 내 prototype
- **플레이**: `zombiehunter/index.html` 브라우저 열기
- **빌드 없음**: 정적 HTML+JS, 번들러 불필요
