# 상세 기능 명세서 (Functional Requirements)

## 1. 유저 스토리 및 핵심 기능 명세

### 1.1 게임 시작 및 설정 (Game Initializer)
* 사용자는 메인 화면에서 '초급', '중급', '고급' 난이도를 선택할 수 있다.
* 난이도 버튼 클릭 즉시 기존 판을 클리어하고 새로운 유효 스도쿠 보드를 생성한다.
* 로컬 스토리지에 진행 중인 게임 데이터가 있다면 '이어서 하기' 버튼을 활성화한다.

### 1.2 플레이 로직 (In-Game Logic)
* **셀 선택:** 스도쿠 보드의 특정 칸을 클릭/터치하면 해당 셀이 선택 상태가 되며, 동일 행/열/3x3 블록의 배경색이 옅게 하이라이트된다.
* **숫자 입력:** 가상 키패드 또는 키보드(1-9)를 통해 숫자를 입력한다. 빈칸일 경우에만 입력이 허용되며, 시스템 초기 고정 값은 수정할 수 없다.
* **메모 모드 (Pencil Marks):** 메모 토글이 켜진 상태에서 숫자를 입력하면, 해당 셀 내부에 작은 3x3 형태의 숫자가 토글 방식으로 기록/삭제된다.
* **오류 검증 (Validation):** 숫자가 입력될 때마다 가로, 세로, 3x3 블록 내 중복 여부를 실시간 검사하여 중복 시 빨간색 스타일을 부여한다.

### 1.3 유틸리티 기능 (Utilities)
* **실행 취소 (Undo):** 사용자의 입력 행동(숫자 입력, 지우기)을 스택에 기록하여 이전 상태로 되돌릴 수 있다.
* **지우기 (Erase):** 선택된 유저 입력 셀의 값을 초기화한다.
* **힌트 (Hint):** 현재 선택된 빈칸에 올바른 정답 숫자를 자동으로 채워준다. 게임당 최대 3회로 제한된다.
