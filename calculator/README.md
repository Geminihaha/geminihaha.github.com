Calculator — simple web calculator

This is a tiny static calculator suitable for GitHub Pages.

Files:
- index.html : single-file app (HTML/CSS/JS) with basic arithmetic, percent, backspace, and clear.

Usage:
- Open index.html in a browser.
- To publish: add this folder to your repository `geminihaha.github.com` and push to GitHub Pages.
