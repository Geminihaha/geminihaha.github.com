# 👻 Ghost Hero — 고스트 히어로

뱀파이어 서바이버(Vampire Survivors) 스타일의 웹 서바이벌 액션 게임.
밀려오는 유령 군단으로부터 **15분간 살아남으면 승리**한다.

- **플레이**: `ghosthero/index.html`을 브라우저로 열기 (GitHub Pages: `/ghosthero/`)
- **구성**: 단일 HTML 파일 (외부 라이브러리 없음, 순수 Canvas 2D + Vanilla JS)
- **제작일**: 2026-07-06

---

## 조작

| 입력 | 동작 |
|---|---|
| WASD / 방향키 | 이동 (PC) |
| 화면 터치 드래그 | 가상 조이스틱 이동 (모바일) |
| P / ESC / ⏸ 버튼 | 일시정지 |
| 공격 | 전부 자동 (무기별 쿨다운) |

## 게임 규칙 (뱀파이어 서바이버 공식)

1. 적을 처치하면 **경험치 보석**이 드랍되고, 획득 범위 안에 들어오면 자동 흡수된다.
2. 레벨업 시 게임이 멈추고 **무작위 3개 강화 중 1개를 선택**한다 (신규 무기 / 무기 강화 / 패시브).
3. 무기 최대 5종 + 패시브 최대 5종 보유, 무기는 8레벨·패시브는 5레벨까지 강화 가능.
4. 시간이 흐를수록 적의 종류·체력·물량이 증가한다.
5. 연속 레벨업 시 선택 기회가 큐에 쌓여 순차적으로 모두 제공된다.

## 캐릭터 (3종)

| 캐릭터 | 특성 | 시작 무기 |
|---|---|---|
| 🛡️ **고스트 나이트** | 체력 130 (높음), 방어력 +1 | ⚔️ 유령검 |
| 🔮 **레이스 메이지** | 공격력 +20%, 쿨다운 -12%, 체력 85 (낮음) | 🔵 영혼 화살 |
| 🏹 **팬텀 레인저** | 이동속도 200 (높음), 획득 범위 +50% | 🌀 유령 표창 |

## 무기 (8종, 각 8레벨)

VS 원작의 무기 포지션을 재해석했다.

| 무기 | 동작 | 원작 포지션 |
|---|---|---|
| ⚔️ 유령검 | 바라보는 방향 부채꼴 베기, 4레벨부터 후방 동시 공격 | 채찍 (Whip) |
| 🔵 영혼 화살 | 가장 가까운 적에게 마력탄 발사, 발사 수 증가 | 마법 지팡이 (Magic Wand) |
| 🌀 유령 표창 | 이동 방향으로 관통 표창 투척 | 나이프 (Knife) |
| 🔥 성화 폭탄 | 폭탄을 던져 지속 피해 화염 지대 생성 | 성수 (Santa Water) |
| ⚡ 원혼의 번개 | 화면 내 무작위 적에게 낙뢰 + 주변 스플래시 | 번개 반지 (Lightning Ring) |
| 💜 영혼 결계 | 플레이어 주위 상시 오라, 지속 피해 + 넉백 | 마늘 (Garlic) |
| 👻 수호 영혼 | 회전 구체가 주위를 돌며 타격·넉백 | 킹 바이블 (King Bible) |
| 🪓 사신의 낫 | 포물선 투척, 관통·고위력 | 도끼 (Axe) |

## 패시브 (6종, 각 5레벨)

| 패시브 | 효과 (레벨당) |
|---|---|
| 💪 유령의 힘 | 모든 피해량 +10% |
| ❤️ 강령의 심장 | 최대 체력 +25 (획득 시 즉시 회복 포함) |
| 👟 유령 발걸음 | 이동속도 +8% |
| ⏳ 시간 왜곡 | 공격 쿨다운 -7% |
| 🧲 영혼 자석 | 경험치 획득 범위 +30% |
| 🛡️ 원혼 갑주 | 받는 피해 -1 |

## 적 / 난이도 진행

- **일반 몬스터 7종**: 👻 유령(0:00) → 🦇 박쥐(0:50) → 🧟 좀비(2:00) → 💀 해골(3:50) → 👹 오거(6:00) → 🧛 뱀파이어(8:00) → 😈 데몬(10:20) 순으로 해금. 최근 해금된 적일수록 스폰 가중치가 높다.
- **스케일링**: 적 체력은 분당 +45%, 접촉 피해는 분당 +10%, 스폰 간격은 1.25초 → 0.28초까지 단축, 스폰 물량은 75초마다 +1.
- **엘리트**: 75초마다 등장 (체력 9배, 대형, 금테). 처치 시 대량 경험치 + 회복/자석 아이템 드랍 확률 50%.
- **보스 3연전** (전용 체력바 표시):
  - 5:00 👹 지옥문지기 오거 로드 (HP 2,600)
  - 10:00 🐉 망령 드래곤 (HP 7,500)
  - 14:00 💀 사신 - 최후의 심판 (HP 16,000)
- **15:00 생존 시 승리** (VICTORY 화면).

## 드랍 아이템

- 🍗 **치킨**: 최대 체력의 35% 회복 (일반 몹 0.8%, 엘리트/보스 확정·확률 드랍)
- 🧲 **자석**: 필드의 모든 경험치 보석 즉시 흡수

## 구현 특징

- **렌더링**: Canvas 2D, devicePixelRatio 대응, 카메라(플레이어 중앙 고정) + 화면 흔들림, 타일 해시 기반 결정적 배경 장식(🪦🌲🕸️), 오프스크린 컬링
- **연출**: 피격 플래시·넉백, 데미지 플로팅 텍스트, 베기/낙뢰/처치 이펙트, 무적시간 점멸
- **성능**: 적 최대 320마리 캡, 경험치 보석 350개 초과 시 오래된 보석 병합, 멀리 벗어난 잡몹은 플레이어 주변으로 재배치
- **기록**: 최고 생존 기록(시간/처치/캐릭터/승리 여부)을 localStorage에 저장, 시작 화면에 표시
- **모바일**: 뷰포트 고정, 터치 가상 조이스틱, 반응형 UI
